# EXScript

A simply overloaded "npm run"!

# Installation

```
npm i -g exscript
```

# Usage

[] = optional, <> = required

#### Editing Scripts
$NODE_Global_Pacakages/exscript/.scripty/scripts.json is where we store scripts.

```
exscript config [-e]
// (-e opens in explorer)
```
#### Running Scripts
```
exscript -r <name|default:default>
// default: echo \"hello\"
```

#### Reloading Config

```
exscript reload
```
